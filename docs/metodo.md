# Método SEIS/OITO

O **Método SEIS/OITO** identifica padrões numéricos ocultos no Manuscrito Voynich.  
A lógica central é que certas palavras seguem a estrutura: primeira e quarta letra iguais; segunda e terceira diferentes, mas com “i” em comum.  

Esse padrão permite mapear tokens voynichianos para números por extenso em occitano medieval.  
Além disso, variantes gráficas são reclassificadas como **CT + marcador**, onde o núcleo corresponde ao número 4 (quatre) e o marcador diferencia as variantes (H, P, K, F, Z/Q).
