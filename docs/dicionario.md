# Dicionário paralelo (1–10)

Este dicionário mostra os números por extenso em occitano medieval, seus candidatos voynichianos e robustez.

- **1 (un)** → candidatos fracos  
- **2 (dos)** → candidatos fracos  
- **3 (tres)** → candidatos médios  
- **4 (quatre)** → CT + marcador (H, P, K, F, Z/Q)  
- **5 (cinc)** → dain  
- **6 (seis)** → daiin  
- **7 (sèt)** → shol / shor / shey  
- **8 (uèch)** → qokain / qokar / qotor  
- **9 (nòu)** → okol / qokol  
- **10 (detz)** → dy / edy
