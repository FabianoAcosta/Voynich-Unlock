# Fluxo de Trabalho

1. **Entrada:** fólios transcritos do manuscrito Voynich.
2. **Processamento:**
   - Detector SEIS/OITO → identifica candidatos a números.
   - Dicionário paralelo → substitui tokens por números.
   - Reclassificação CT → aplica marcadores (H, P, K, F, Z/Q).
3. **Saída:** corpus paralelo (Original / Numérico / PT / EN).
4. **Validação:** testes unitários em `tests/`.
5. **Publicação:** atualização contínua da documentação e corpus.
